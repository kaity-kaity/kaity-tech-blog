# 概要

- パーソナルトレー/パーソナルトレーナーを雇用するジムへの顧客管理システム
- トレーニングに特化

# 課題目的

### トレーナー(クライアント)課題解決

- 集客困難
- 顧客管理の簡易化
- カスタマイズ性(IT リテラシーが低くて、トレーニング知識が豊富なトレーナーでもカスタマイズ)

### トレーニー(顧客)

- トレーニングに対するモチベーションの向上

# 使用技術

### フロントエンド

- Next.js
- Material UI
- その他ライブラリ検討

### バックエンド

- Node.js(express)

### DB

- MongoDB or MySQL or Firestore (検討中)

### クラウドサービス

- AWS or GCP(検討中)

### クラウドストレージ

- Cloudinary
