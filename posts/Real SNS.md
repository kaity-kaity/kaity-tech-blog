# 概要

- オリジナル SNS アプリ

# 開発目的

- フルスタック(MERN)での開発経験

# 使用技術

### フロントエンド

- React.js
- Redux
- MaterialUI

### バックエンド

- Node.js(express)

### DB

- MongoDB

### デプロイ

- AWS EC2 (デプロイ)

### クラウドストレージ
